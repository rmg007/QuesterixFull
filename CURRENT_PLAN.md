## When a Plan is Active

This file will show:

```
Active Plan:  001-feature-name
Status:       In Progress
Phase:        Implementation
File:         PLANS/001-feature-name.md
```

**Agents:** Read this file first (5 seconds). Then open the full plan in `PLANS/`.

---

## To Create a New Plan

Use: `/plan <slug> <problem>`

Example:
```
/plan add-dark-mode "Add dark mode toggle to settings"
/plan fix-auth-race "Fix race condition in token refresh"
```
